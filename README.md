# hotelgarden.pl

## Install dependencies

```
$ npm install
```

## Run for development
```
$ npm run dev
```

## Building
```
$ npm run build
```

## Deployment
Before deploying:
* run `npm run firebase-login` if you're not logged in to firebase.
* build
```
$ npm run deploy
```

## Other

### Generate index.html
```
$ npm run generate-index
```

### Bundle lodash
```
$ npm run bundle-lodash
```
